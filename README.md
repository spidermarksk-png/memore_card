Memory card

Эта приложение создано для помощи в подготовке к зачету по физике для 7 класса
 
Приложение написано на Python'е, используя модули PyQt5 и random.

Это приложение — викторина. Основная идея — задавать пользователю вопросы с несколькими вариантами ответов, проверять правильность выбранного варианта и показывать результаты.
Вначале создаётся модель данных — класс Question, каждый объект которого содержит вопрос и четыре варианта ответов, среди которых один правильный. Эти вопросы собираются в список. Интерфейс состоит из основного окна, на котором отображается текст вопроса, группа радиокнопок с вариантами ответов и кнопки для перехода к следующему вопросу и проверки ответа. При запуске программа выбирает случайный вопрос из списка и перемешивает варианты ответов, чтобы правильный ответ не всегда был на одном месте. Пользователь выбирает вариант и нажимает кнопку "Ответить"; программа проверяет, выбран ли правильный ответ, выводит сообщение о правильности или неправильности, обновляет статистику и выводит результат в консоль. После этого при нажатии "Следующий вопрос" программа случайным образом выбирает новый вопрос, очищает выбранные варианты, и цикл повторяется. Варианты ответов перемешиваются каждый раз для разнообразия, а статистика показывает количество правильных и всех ответов. 


Такой подход обеспечивает интерактивное обучение или тренажёр, позволяя пользователю погружаться в вопросы по физике и закреплять знания в игровой форме.
